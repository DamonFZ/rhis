# RHIS 功能清单

> 本文档记录所有已完成/开发中的功能模块，每次更新或完成新功能后需同步修改此文件。

---

## 模块一：组织架构管理

### 1.1 部门管理

**功能描述：** 支持多级部门（科室）的创建与管理，采用树形层级结构。

| 功能点 | 状态 | 说明 |
| ------ | ---- | ---- |
| 部门列表展示 | ✅ 已完成 | 按排序字段展示所有部门，支持分页 |
| 新建顶级部门 | ✅ 已完成 | 不选父部门即创建为顶级部门 |
| 新建子部门 | ✅ 已完成 | 选择父部门后创建子部门，支持多级嵌套 |
| 编辑部门 | ✅ 已完成 | 修改部门名称、编码、层级、排序、状态、描述 |
| 删除部门 | ✅ 已完成 | 支持单个删除和批量删除 |
| 部门搜索 | ✅ 已完成 | 按部门名称、编码搜索 |
| 父部门筛选 | ✅ 已完成 | 按父部门过滤列表 |
| 状态筛选 | ✅ 已完成 | 筛选启用/禁用状态的部门 |
| 编辑时排除自身 | ✅ 已完成 | 编辑部门时不能选择自己作为父部门 |
| 层级自动设置 | ✅ 已完成 | 创建时层级默认为1 |
| 排序管理 | ✅ 已完成 | 通过 sort 字段控制部门显示顺序 |

**字段：**
- `parent_id` - 父部门ID（默认0为顶级部门）
- `name` - 部门名称（必填，最大100字符）
- `code` - 部门编码
- `level` - 层级（默认1）
- `sort` - 排序（默认0）
- `status` - 状态（启用/禁用）
- `description` - 部门描述

---

### 1.2 员工管理

**功能描述：** 管理系统员工信息，支持员工与部门的多对多关联。

| 功能点 | 状态 | 说明 |
| ------ | ---- | ---- |
| 员工列表展示 | ✅ 已完成 | 展示所有员工信息，支持分页 |
| 新建员工 | ✅ 已完成 | 创建新员工，密码在创建时必填 |
| 编辑员工 | ✅ 已完成 | 修改员工信息，密码选填（不填则不修改） |
| 删除员工 | ✅ 已完成 | 支持单个删除和批量删除 |
| 员工搜索 | ✅ 已完成 | 按姓名、邮箱、职位搜索 |
| 职位管理 | ✅ 已完成 | 记录员工职位名称 |
| 邮箱非必填 | ✅ 已完成 | 邮箱字段改为可选 |

**字段：**
- `name` - 姓名（必填）
- `email` - 邮箱（可选，唯一）
- `password` - 密码（创建时必填，编辑时选填）
- `position` - 职位

---

### 1.3 部门员工关联管理

**功能描述：** 在部门详情页管理该部门的员工，支持关联和创建两种模式。

| 功能点 | 状态 | 说明 |
| ------ | ---- | ---- |
| 部门员工列表 | ✅ 已完成 | 在部门详情页展示该部门所有员工 |
| 关联已有员工 | ✅ 已完成 | 从系统中选择已有员工关联到当前部门 |
| 创建新员工并关联 | ✅ 已完成 | 创建新员工的同时关联到当前部门 |
| 移除员工 | ✅ 已完成 | 将员工从当前部门移除（不删除员工本身） |
| 删除员工 | ✅ 已完成 | 彻底删除员工（同步解除所有部门关联） |
| 批量移除 | ✅ 已完成 | 批量将员工从当前部门移除 |
| 主部门标记 | ✅ 已完成 | 标记员工的主部门 |
| 编辑主部门状态 | ✅ 已完成 | 编辑员工时修改主部门状态 |
| 创建时设置主部门 | ✅ 已完成 | 创建员工时设置是否为主部门 |
| 关联时默认非主部门 | ✅ 已完成 | 关联已有员工时默认非主部门 |
| 排除已关联员工 | ✅ 已完成 | 关联员工时只显示未关联到当前部门的员工 |

**字段：**
- `name` - 姓名
- `email` - 邮箱
- `password` - 密码
- `position` - 职位
- `is_primary` - 是否主部门

**功能描述：** 管理客户购买的套餐包，记录剩余次数等资产信息。

| 功能点 | 状态 | 说明 |
| ------ | ---- | ---- |
| 套餐包列表 | ✅ 已完成 | 在客户详情页展示该客户所有套餐包 |
| 新建套餐包 | ✅ 已完成 | 为客户添加新套餐包 |
| 编辑套餐包 | ✅ 已完成 | 修改套餐包信息 |
| 删除套餐包 | ✅ 已完成 | 删除套餐包（级联删除关联消费记录） |
| 状态管理 | ✅ 已完成 | 标记套餐包为 active/completed |
| 自动标记完成 | ✅ 已完成 | 剩余次数≤0 时自动标记为已完成 |
| 次数扣减 | ✅ 已完成 | 创建消费记录时自动扣减剩余次数 |
