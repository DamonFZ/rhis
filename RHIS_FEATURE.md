## 模块一：组织架构管理

### 1.1 部门管理

**功能描述：** 支持多级部门（科室）的创建与管理，采用树形层级结构。

| 功能点 | 状态 | 说明 |
| ------ | ---- | ---- |
| 部门列表展示 | ✅ 已完成 | 按排序字段展示所有部门，支持分页 |
| 新建顶级部门 | ✅ 已完成 | 不选父部门即创建为顶级部门 |
| 新建子部门 | ✅ 已完成 | 选择父部门后创建子部门，支持多级嵌套 |
| 编辑部门 | ✅ 已完成 | 修改部门名称、编码、层级、排序、状态、描述 |
| 删除部门 | ✅ 已完成 | 支持单个删除和批量删除 |
| 部门搜索 | ✅ 已完成 | 按部门名称、编码搜索 |
| 父部门筛选 | ✅ 已完成 | 按父部门过滤列表 |
| 状态筛选 | ✅ 已完成 | 筛选启用/禁用状态的部门 |
| 编辑时排除自身 | ✅ 已完成 | 编辑部门时不能选择自己作为父部门 |
| 层级自动设置 | ✅ 已完成 | 创建时层级默认为1 |
| 排序管理 | ✅ 已完成 | 通过 sort 字段控制部门显示顺序 |

**字段：**
- `parent_id` - 父部门ID（默认0为顶级部门）
- `name` - 部门名称（必填，最大100字符）
- `code` - 部门编码
- `level` - 层级（默认1）
- `sort` - 排序（默认0）
- `status` - 状态（启用/禁用）
- `description` - 部门描述

### 5.2 康复成效对比系统

**功能描述：** 对比同一客户的多次评估记录（最多5条），展示数据宽表，总成效基于首尾记录计算差值，帮助康复师直观了解康复效果。

| 功能点 | 状态 | 说明 |
| ------ | ---- | ---- |
| 批量选择评估记录 | ✅ 已完成 | 在评估列表支持选择2-5条记录 |
| 对比成效按钮 | ✅ 已完成 | 点击后跳转到对比页面 |
| 数据宽表展示 | ✅ 已完成 | 展示所有选中记录的完整数据 |
| 首尾记录差值计算 | ✅ 已完成 | 基于日期最前和最后记录计算差值 |
| 基础体测对比 | ✅ 已完成 | 身高、体重、BMI、体脂率的差值计算 |
| 详细围度对比 | ✅ 已完成 | 胸围、腰围、臀围、左右臂、左右大腿围的差值 |
| 柔软度评估对比 | ✅ 已完成 | 对比躯干、腘绳肌、髂腰肌群等的变化 |
| 体态评估-侧面对比 | ✅ 已完成 | 对比头部、颈椎、肩胛骨等侧面的变化 |
| 体态评估-背面对比 | ✅ 已完成 | 对比颈椎、肩部、肩胛骨等背面的变化 |
| 成效可视化 | ✅ 已完成 | 数值变好（减小）用绿色向下箭头，变差用红色向上箭头 |
| 定性对比展示 | ✅ 已完成 | 非数值对比显示"有变化"或"无变化" |
| 打印报告 | ✅ 已完成 | 支持一键打印，使用Tailwind print:hidden优化打印样式 |
| 返回功能 | ✅ 已完成 | 支持返回上一页 |

**对比内容：**
- 数据宽表：所有选中记录的完整数据（2-5条）
- 总成效：基于首尾记录计算差值
- 基础体测：身高、体重、BMI、体脂率
- 详细围度：胸围、腰围、臀围、左右臂、左右大腿围
- 柔软度评估：躯干、腘绳肌、髂腰肌群、股四头肌、小腿肌群、肩部
- 体态评估-侧面：头部、颈椎、肩胛骨、胸椎、腰椎、骨盆、膝关节
- 体态评估-背面：颈椎、肩部、肩胛骨、胸腰椎、骨盆、膝关节、足弓
- 数值对比：绿色（&darr;）表示好转，红色（&uarr;）表示变差
- 定性对比：显示"有变化"或"无变化"
